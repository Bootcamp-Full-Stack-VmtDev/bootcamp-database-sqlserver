## 📑 Descripción

Este repositorio contiene todos los ejercicios, prácticas y proyecto de SQL Server del Bootcamp **Full Stack Developer: Impulsa tu Futuro**.  

Incluye desde prácticas de DDL, DML, joins, agregaciones, hasta un proyecto final basado en una plataforma de cursos tipo Udemy.

## 🏷 Módulos y contenido

### 2️⃣ Arquitectura de datos con DDL
📄 Contiene prácticas para modelado y creación de estructuras de base de datos:
- `Practica - CloneDiscord.sql` – Clon de la estructura de Discord.  
- `Practica - TalentInsights.sql` – Práctica de modelado y queries.  
- `TalentInsights.sql` – Script completo de Talent Insights.

### 3️⃣ DML – Dando vida a los datos
💾 Scripts para insertar, actualizar y manipular datos:  
- `LeaderboardBootcamp.sql` – Ejercicios de inserción y actualización de datos de práctica.

### 4️⃣ Joins y agregación
🔗 Prácticas avanzadas con joins, agregaciones y análisis de datos:  
- `AdventureWorks.sql` – Prácticas con la base AdventureWorks.  
- `Practica - AdventureWorks.sql` – Ejercicios de joins y queries.  
- `AdventureWorksLT2022.bak` – Backup de la base de datos AdventureWorks LT 2022.

### 5️⃣ Proyecto final – Plataforma Udemy
🎯 Proyecto final del Bootcamp, con base de datos completa, scripts y consultas:  
- `diagrama-proyecto.svg` – Diagrama entidad-relación.  
- `Proyecto-Udemy-DDL.sql` – Script DDL completo.  
- `Proyecto-Udemy-DML.sql` – Script de datos.  
- `Proyecto-Udemy-Ejercicios.sql` – Consultas del proyecto.  
- `Proyecto_Udemy.pdf` – Documento guía y presentación.  
- `UdemyDatabase.sql` – Script completo de la base de datos.  
- `udemy_infography.jpeg` – Infografía resumen del proyecto.

📌 Para más detalles sobre el proyecto, revisar el [`README.md`](./proyecto-sql-server/README.md) dentro de la carpeta del proyecto.
